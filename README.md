# 🎨 flight-finder-front

Aplicación frontend desarrollada en **Angular** para la plataforma Flight Finder App.

Este proyecto permite a los usuarios interactuar con el sistema de búsqueda de vuelos mediante una interfaz moderna, dinámica y responsive.

---

## 📌 Descripción

El frontend consume los microservicios del backend a través de API REST y permite:

- 🔎 Buscar vuelos por origen, destino y fecha
- 💲 Visualizar vuelos más económicos
- 👤 Registro de usuarios
- 🔐 Inicio de sesión
- 📊 Visualización estructurada de resultados
